1、安装nodejs 6.0以上
2、安装淘宝镜像(安装后可用cnpm 来装各模块)
   npm install -g cnpm --registry=https://registry.npm.taobao.org
3、安装dev全局环境
   cnpm install -g @vue/cli
   cnpm install -g @vue/cli-init
4、npm install 安装完全部模块
5、npm run serve 运行开发模式
6、http://localhost:8085/ 谷歌浏览器打开看到程序界面
7、npm run build  编译一次，生成dist文件夹，可直接发布到线上站点
